About

Grid is a graphical user interface for Kodi, an award-winning free and open source software entertainment hub available on Windows, OSX, Linux, Android and iOS. It aims to provide an elegant and intuitive way of enjoying Kodi's featureset and to extend on it in a sensible way. Grid's design and development process focuses on feature quality rather than quantity with a lot of time and care spent on the little details that matter. To find out more about Kodi click here.

Download

There are different versions of Grid available, which are obtained in different ways. Please refer to the Github project page for installation instructions.

  • Latest release

    Grid is currently in beta stage. Official, stable releases of Grid can be installed through the official Kodi repository when it is made publicly available.

  • Latest beta

    You can get the latest beta release by installing my repository add-on, which can be downloaded here. Please note that these releases are meant for testing and as such they usually contain more bugs. It is not recommended to use this on your primary device.

  • Snapshot build

    These are generated by Github and contain the most recent changes at the time of downloading. They should be considered unstable and very likely to contain bugs or broken / missing functionality. At the moment these builds are not yet publicly available.
